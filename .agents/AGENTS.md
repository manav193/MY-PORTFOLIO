# Custom Behavioral Rules

## Strict Image Placement (Case Studies)
- **1 Image = 1 Location:** Every screenshot may appear ONLY ONE TIME on the entire page.
- **No Duplicates:** Never reuse an image in another section. No decorative duplicates, blurred backgrounds, or floating duplicates.
- **No Cropping/Zooming for Reuse:** Never crop one image into multiple layouts to artificially generate assets.
- **Meaning Match:** The image placement must accurately match the content being discussed (e.g., Homepage -> Hero, Dashboard -> Dashboard section).
- **Graceful Degradation:** If there are fewer screenshots than sections, keep the remaining sections text-only. Do NOT force screenshots into every section.
- **Preferred Priority:** 1. Hero, 2. Overview, 3. Features, 4. Process, 5. Gallery, 6. Final Showcase.
